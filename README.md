# Chrome Addon Setup

## Installera addonet i Chrome

1. Öppna Chrome och gå till:

chrome://extensions/

2. Sätt på **Programmerarläge** (toggle uppe till höger)

3. Klicka på **Läs in opaketerat tillägg**

4. Välj addon-mappen på din dator

---

## Lägg in Discord webhook

1. Öppna din Discord-server  
2. Gå till **Serverinställningar**  
3. Gå till **Integrationer**  
4. Klicka på **Webhook**  
5. Klicka på **Ny webhook**  
6. Kopiera webhook-URL  

Klistra in webhook-URL:en i addonets popup och klicka på **Spara**

---

## Hur vet jag om jag har rätt version?

1. Gå till projektets GitHub-sida  
2. Klicka på **master**  
3. Kontrollera om det finns något nyare  

Om GitHub har nyare ändringar än din lokala version → du behöver uppdatera
